\section{Abstract}
Reconsolidation is a replicable process in neurobiology to treat memory recall that is emotionally disruptive. 
Inspired by a distributive network able to cleanly target a memory and modify it, we propose a way to translate
this process into machine learning to solve the unlearning problem.
